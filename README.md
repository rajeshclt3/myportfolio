# Tech Portfolio Starter (Astro + Tailwind + MDX)

This starter scaffolds an Astro site with Tailwind CSS and MDX-based content and a component-driven architecture suitable for a portfolio (Home, Resume, Work, Contact).

Quick start
1. Install dependencies:
```bash
npm install
```
2. Run dev server:
```bash
npm run dev
```
3. Build for production:
```bash
npm run build
```

Deployment
- Use GitHub Pages, Netlify, or Cloudflare Pages. For GitHub Pages, add a workflow to build and publish the `dist/` output or use the `gh-pages` branch.

Editing content
- Page content is stored in `src/content/*.md` or `src/content/projects/*.md`. Edit via GitHub web UI or a local editor and commit.
